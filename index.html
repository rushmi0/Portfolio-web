UTxO คืออะไร


Bitcoin ใช้รูปแบบการจดบัญชีโดยการใช้ UTxO Model


UTxO (Unspent Transaction Output) คือ เงินที่เรามีอยู่ แต่เรายังไม่ได้ใช้จ่ายเงินนั้นออกไป


Output ของ UTxO หนึ่งเป็นการใช้จ่ายออกไปของคนหนึ่ง แล้วการใช้จ่ายของคนนั้น จะกลายเป็นรายรับของอีกคนหรือเป็น Input ของอีก UTxO ต่อไป

ยกตัวอย่างง่าย ๆ ให้เห็นว่ามันเป็นยังไง


ฟ้ามีเงินอยู่ 1 BTC .. ต้องการจะโอนเงินให้ส้ม 0.5 BTC ฟ้าจะสร้าง ธุรกรรมขึ้นมา ขอเรียกว่า "Tx1" ในขาเข้า UTxO ของฟ้าจะมี 1 BTC


ขาออกจะมี 2 Output ออกมา จากที่เห็นจะ มีเงินของฟ้าโผล่ออกมาจาก UTxO นี้จำนวน 0.5 BTC เป็นเพราะว่า ฟ้าโอนเงิน 0.5 BTC ให้กับส้ม .. พอฟ้าส่ง 0.5 BTC ให้กับส้มแล้ว ฟ้าจะได้เงินทอนให้ตัวเองกลับมา 0.5 BT

การใช้จ่ายแบบ UTxO มันเหมือนกับการใช้เงินสด


การใช้เงินแบบ UTxO จะต้องใช้จ่ายเงินทั้งหมดออกไป แล้วจะได้เงินทอนกลับมา (แอป Wallet จะสร้าง Address สำหรับรับเงินทอนเตรียมไว้อยู่แล้ว)


แบบนี้เราจะเห็นว่าผลรวม UTxO ขาเข้า จะต้องเท่ากับผลรวมของ UTxO ขาออกเสมอ (กรณีตัวอย่างนี้ไม่มีค่า Fee ของจริงจะมี UTxO แตกออกไปเป็นค่า Fee ด้วยครับ)

คุณสมบัติอีกอย่างของ UTxO


UTxO จะมี Input เท่าไหร่ก็ได้ และมี Output เท่าไหร่ก็ได้ เงินโอนเข้ามา 100 Address แล้วออกมา 200 Address ... หรือจะเข้ามา 5 Address ออกมา 5 Address ก็ได้


แต่ผลรวมของเงินในแต่ละ Address ทั้งขาเข้าและขาออก จะเท่ากันนะ เป็นความสามารถหลอมรวม BTC เป็นหน่วยใหญ่และแตกเป็นหน่วยย่อย ๆ ก็ได้


การเคลื่อนไหวของเงิน จากมุมมองคนนอกที่มองเข้ามา จะไม่รู้เลยว่าใครเป็นใคร จะเห็นว่ามีเงินจาก Address หนึ่ง โอนเงินไปที่อีก Address หนึ่ง


Address นั้นอาจเป็นของคนอื่นจริง ๆ หรือเป็นของเจ้าตัวของคนที่โอนเองก็ได้ เพราะเราสามารถสร้าง Address ใหม่ ๆ ออกมารับเงินได้เรื่อย ๆ เลยล่ะ
